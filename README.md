📍 Contact Book

A simple and interactive Contact Book Web App that allows users to store, view contacts efficiently.

Users can add contact details like name, phone number, address, profession, and profile image. The contacts are displayed in a smooth carousel UI, making navigation clean and user-friendly.

This project demonstrates practical frontend development concepts using JavaScript and DOM manipulation.

Built using HTML, CSS, Bootstrap 5, and JavaScript.

🚀 Features

- ✅ Add new contacts using a modal form
- ✅ Form validation with real-time error messages
- ✅ Store contacts using localStorage
- ✅ Display contacts in a Bootstrap carousel
- ✅ Dynamic Rendering
